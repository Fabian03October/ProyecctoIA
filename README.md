# Proyecto YOLO para asistencia a personas con discapacidad visual

Este proyecto detecta objetos en tiempo real utilizando YOLOv8, calcula distancia aproximada y envía mensajes de voz para apoyar a personas con discapacidad visual.

## Estructura
- `data/` -> dataset (imágenes + etiquetas YOLO)
- `src/train.py` -> Entrenamiento
- `src/detect.py` -> Cámara en tiempo real
- `src/distance.py` -> Cálculo de distancia
- `src/voice.py` -> Texto a voz
- `models/` -> Modelos entrenados
